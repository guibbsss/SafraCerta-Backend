SafraCerta – API local
========================

Pré-requisitos
--------------
- Java 17+
- MySQL 8 em execução; crie a base de dados `safracerta` antes de subir a API

Banco de dados
--------------
Por padrão a aplicação usa:
  URL: jdbc:mysql://localhost:3306/safracerta
  Usuário: root
  Senha: (vazia)

Para outro host, usuário ou senha, defina variáveis de ambiente antes de subir a API:
  SPRING_DATASOURCE_URL
  SPRING_DATASOURCE_USERNAME
  SPRING_DATASOURCE_PASSWORD

Subir a API
-----------
Na pasta do backend:

  Windows:  .\mvnw.cmd spring-boot:run
  Linux/mac: ./mvnw spring-boot:run

Rotas úteis (prefixo /api)
--------------------------
- GET  http://localhost:8080/api/health
- POST http://localhost:8080/api/auth/login
       Corpo JSON: {"email":"user@exemplo.com","password":"qualquer"}
       Resposta: token e user (stub de desenvolvimento; substituir depois por auth real.)

O front Angular (npm start) usa apiUrl http://localhost:8080/api – alinhado ao context-path acima.

Documentação OpenAPI (Swagger)
------------------------------
Com a API em execução:
  UI:     http://localhost:8080/api/swagger-ui/index.html
          (atalho: http://localhost:8080/api/swagger-ui.html)
  JSON:   http://localhost:8080/api/v3/api-docs

Novos controllers e métodos @RestController aparecem na UI automaticamente (SpringDoc).
